# DAO提案系统 - 用户指南

## 🎯 系统解决的核心问题

### 问题1：DAO提案理解困难
**痛点：**
- DAO成员提交的提案往往很长、技术性强
- 普通成员难以快速理解提案的核心内容和影响
- 投票时缺乏足够信息，只能凭感觉投票

**解决方案：**
- AI自动分析提案，生成简洁易懂的摘要
- 多维度风险评估（技术、财务、治理、执行）
- 清晰的决策建议（支持/反对/修改建议）

### 问题2：提案分析成本高
**痛点：**
- 人工分析提案需要专业知识，成本高
- 小DAO无法负担专业分析师
- 分析结果可能带有主观偏见

**解决方案：**
- 去中心化AI网络自动分析，成本极低
- 多个节点独立分析，避免单点偏见
- 社区拥有AI模型，不受中心化公司控制

### 问题3：提案数据分散，难以追溯
**痛点：**
- 提案散落在论坛、Discord、Snapshot等平台
- 历史提案难以查找和对比
- 提案执行结果无法追踪

**解决方案：**
- 所有提案统一存储在IPFS，永久保存
- 链上记录提案哈希，不可篡改
- 提案状态全程追踪（提交→分析→投票→执行）

### 问题4：缺乏激励机制
**痛点：**
- 提案提交者、分析者、投票者缺乏激励
- 优质提案和贡献得不到奖励
- 社区参与度低

**解决方案：**
- 代币奖励机制，贡献者获得$PROPOSAL代币
- 数据贡献、节点服务、投票参与都有奖励
- 形成正向经济循环

## 👥 用户角色和使用场景

### 角色1：DAO成员（提案提交者）

**使用场景：**
小明是某个DeFi DAO的成员，他有一个改进协议的想法，但不知道如何写正式提案。

**使用流程：**
1. **打开系统首页**
   - 看到简洁的界面："提交你的DAO提案"
   - 点击"创建提案"按钮

2. **填写提案内容**
   - 在文本框中输入想法（可以是草稿，不需要完美）
   - 例如："我建议将协议的手续费从0.3%降低到0.25%，这样可以吸引更多用户"
   - 可以上传附件（图片、文档等）

3. **提交提案**
   - 点击"提交"按钮
   - 连接Solana钱包（Phantom/Solflare）
   - 确认交易（费用约$0.00025，几乎免费）

4. **等待AI分析**
   - 系统自动上传提案到IPFS（去中心化存储）
   - 多个AI节点开始分析提案
   - 约30秒后，分析结果自动生成

5. **查看分析结果**
   - 看到AI生成的摘要：
     ```
     核心内容：建议将协议手续费从0.3%降至0.25%
     预期影响：可能吸引更多用户，但会减少协议收入
     风险评估：财务风险-中等，执行风险-低
     建议：支持，但建议先进行小规模测试
     ```
   - 看到详细的风险分析报告
   - 看到与历史相似提案的对比

6. **获得奖励**
   - 提交提案后，自动获得10个$PROPOSAL代币奖励
   - 如果提案质量高，获得额外奖励

**用户价值：**
- ✅ 无需专业知识，草稿也能提交
- ✅ 快速获得专业分析，了解提案影响
- ✅ 获得代币奖励，激励参与

### 角色2：推理节点运营者

**使用场景：**
小李有GPU服务器，想通过提供AI推理服务赚取代币。

**使用流程：**
1. **注册推理节点**

   - 配置服务器信息（GPU型号、算力等）

2. **运行推理服务**
   - 安装节点软件（Docker容器）
   - 配置LLM模型（Llama 3/Mistral等）
   - 节点自动接收推理任务

3. **提供服务**
   - 当有提案提交时，系统自动分发推理任务
   - 节点分析提案，生成摘要和风险评估
   - 提交推理结果到链上

4. **获得奖励**
   - 每次推理服务获得0.1个$PROPOSAL代币
   - 推理质量高（准确率>90%），额外奖励50%
   - 响应速度快（<2秒），额外奖励20%
   - 每月稳定运行，获得稳定性奖励

5. **查看统计**
   - 节点仪表板显示：
     - 总推理次数：1,234次
     - 平均准确率：92%
     - 总收益：156.7 $PROPOSAL
     - 节点排名：第3名

**用户价值：**
- ✅ 闲置GPU资源变现
- ✅ 稳定被动收入
- ✅ 参与去中心化AI网络建设

### 角色3：DAO投票者

**使用场景：**
小王是DAO成员，需要投票决定是否通过某个提案。

**使用流程：**
1. **查看提案列表**
   - 打开"提案列表"页面
   - 看到所有待投票提案
   - 每个提案卡片显示：
     - 提案标题和摘要
     - AI分析的风险等级（低/中/高）
     - 支持/反对建议
     - 投票截止时间

2. **查看提案详情**
   - 点击感兴趣的提案
   - 看到完整提案内容（从IPFS加载）
   - 看到AI生成的详细分析：
     - 核心内容摘要
     - 风险评估（技术、财务、治理、执行）
     - 影响分析（对社区、代币、技术栈）
     - 决策建议
     - 与历史提案的对比

3. **做出决策**
   - 基于AI分析和自己的判断
   - 点击"支持"或"反对"按钮
   - 连接钱包确认投票

4. **追踪结果**
   - 投票后，提案状态更新
   - 可以看到实时投票统计
   - 提案通过后，追踪执行情况

**用户价值：**
- ✅ 快速理解复杂提案
- ✅ 基于数据做决策，而非凭感觉
- ✅ 节省研究时间，提高投票质量
